## 🏁 Quick Start

1. **Clone the repo**  
   ```bash
   git clone https://github.com/arqam66/smiu_room4.git
   cd smiu_room4
   ```

2. **Install dependencies**  
   ```bash
   pnpm install  # or `npm install`
   ```

3. **Run the dev server**  
   ```bash
   pnpm dev
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser.
